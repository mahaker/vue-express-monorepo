# vue-express-monorepo

This is a sample project that build by lerna(monorepo).

**Packages**

- frontend-app
   - Vue3(vite) app.
- server-app
   - express and TypeScript app. Build by esbuild

## How to run

```shell
# Clone repository
git clone https://github.com/mahaker/esbuild-gas-plugin.git
npm install

# Bootstrap
npm run bootstrap
```
